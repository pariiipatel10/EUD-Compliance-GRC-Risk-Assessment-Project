# EUD Compliance & GRC Risk Assessment Project

## Overview
This project focuses on Endpoint User Device (EUD) Compliance, Governance Risk & Compliance (GRC), SOC Operations, and Vulnerability Management.

The project analyses endpoint security gaps, compliance failures, and cybersecurity risks associated with missing security controls such as:
- EDR
- SCCM
- BitLocker
- Antivirus
- Patch Management

## Key Objectives
- Identify endpoint compliance gaps
- Analyse cybersecurity risks
- Understand SOC monitoring
- Perform vulnerability assessment
- Recommend mitigation strategies

## Technologies & Concepts Used
- GRC Framework
- Risk Assessment
- Vulnerability Management
- SOC Operations
- SIEM Concepts
- Endpoint Security
- Compliance Analysis

## Project Modules
1. Risk Identification
2. Vulnerability Assessment
3. Endpoint Compliance Analysis
4. Security Gap Analysis
5. Risk Heat Map
6. Security Recommendations

## Skills Demonstrated
- Cybersecurity Analysis
- GRC Understanding
- Risk Management
- Technical Documentation
- Security Monitoring Concepts
