# EUD Compliance & GRC Risk Assessment Project

## Overview
This project focuses on Endpoint User Device (EUD) Compliance, Governance Risk & Compliance (GRC), SOC Operations, and Vulnerability Management.

The project analyses endpoint security gaps, compliance failures, and cybersecurity risks associated with missing security controls such as:
- EDR
- SCCM
- BitLocker
- Antivirus
- Patch Management

## Key Objectives
- Identify endpoint compliance gaps
- Analyse cybersecurity risks
- Understand SOC monitoring
- Perform vulnerability assessment
- Recommend mitigation strategies
